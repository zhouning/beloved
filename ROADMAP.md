# 项目路线图（Roadmap）

> AI编程启蒙课 —— 写给初中生的 Claude Code 入门教程

## 当前状态

- 7 节课程内容已全部完成
- VitePress 站点基础框架已搭建
- 自定义主题样式已配置
- 本地开发环境可正常运行

---

## 第一阶段：部署上线（v1.0）

**目标**：让课程可以通过网址访问，任何人都能在线学习。

- [ ] 配置 GitHub Pages 自动部署（GitHub Actions）
- [ ] 设置自定义域名（可选）
- [ ] 添加网站 favicon 和 Open Graph 元信息（分享时显示标题和预览图）
- [ ] 检查移动端适配，确保手机上阅读体验良好
- [ ] 全站内容校对（错别字、链接、格式）

---

## 第二阶段：体验优化（v1.1）

**目标**：提升学生的学习体验，让课程更好用。

- [ ] 添加搜索功能（VitePress 内置 local search）
- [ ] 为每节课添加"预计用时"标注
- [ ] 添加课程进度导航（如"你已完成 3/7 课"的视觉提示）
- [ ] 优化代码块样式，添加一键复制按钮的提示说明
- [ ] 添加页面底部的"常见问题 FAQ"页面
- [ ] 添加"课程反馈"入口（链接到问卷或 GitHub Discussions）

---

## 第三阶段：资源补充（v1.2）

**目标**：提供配套学习素材，降低学习门槛。

- [ ] 为每节课制作"成果示例截图"，让学生知道完成后的效果
- [ ] 提供每节课的完整示例代码（可下载的 zip 或 GitHub 仓库链接）
- [ ] 编写"家长/老师指南"页面（课程设计理念、如何辅导）
- [ ] 制作"常见报错排查"速查表（学生最容易遇到的问题）
- [ ] 添加"术语表"页面（HTML、CSS、API 等概念的简明解释）

---

## 第四阶段：互动增强（v2.0）

**目标**：从静态文档进化为互动学习平台。

- [ ] 嵌入在线代码编辑器（如 StackBlitz / CodeSandbox），支持在网页内直接运行代码
- [ ] 为每节课添加小测验（选择题），巩固知识点
- [ ] 添加"学生作品展示墙"页面，收录优秀学生作品
- [ ] 接入评论系统（如 Giscus），允许学生交流讨论
- [ ] 制作关键步骤的 GIF 动图或短视频演示

---

## 第五阶段：内容扩展（v3.0）

**目标**：丰富课程体系，覆盖更多学习路径。

- [ ] **进阶篇**（第8-12课）：Python 入门、数据可视化、简易聊天机器人等
- [ ] **专题篇**：针对特定兴趣的独立教程（如"用 AI 做像素画"、"用 AI 写小说"）
- [ ] 配套视频教程（录屏 + 讲解，发布到 B站等平台）
- [ ] 多语言支持（英文版），扩大受众范围
- [ ] 探索与学校/机构合作，作为校本课程试点

---

## 里程碑时间线（建议）

| 阶段 | 内容 | 建议周期 |
|------|------|---------|
| v1.0 | 部署上线 | 1-2 周 |
| v1.1 | 体验优化 | 2-3 周 |
| v1.2 | 资源补充 | 3-4 周 |
| v2.0 | 互动增强 | 1-2 月 |
| v3.0 | 内容扩展 | 长期持续 |

---

## 成功指标

- **访问量**：月独立访客数
- **完课率**：从第1课学到第7课的学生比例
- **作品数**：学生提交/展示的作品数量
- **反馈评分**：学生和家长的满意度
- **社区活跃度**：讨论区发帖和互助数量

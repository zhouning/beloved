# 项目路线图（Roadmap）

> AI编程启蒙课 —— 写给初中生的 Claude Code 入门教程

## 当前状态

- 7 节课程内容已全部完成
- 网站已部署上线：[ai4junior.com](https://www.ai4junior.com/)
- v1.0 ~ v2.0 阶段全部完成（仅剩截图和视频素材待补充）
- 互动功能已上线：小测验、评论系统、代码预览、作品展示墙
- 配套资源已完善：家长指南、报错排查、术语表、示例代码

## 第三阶段：资源补充（v1.2） 🔶 基本完成

**目标**：提供配套学习素材，降低学习门槛。

- [ ] 为每节课制作"成果示例截图"，让学生知道完成后的效果（需手动截图）
- [x] 提供每节课的完整示例代码（6 个独立 HTML 文件，浏览器直接运行）
- [x] 编写"家长/老师指南"页面（课程设计理念、如何辅导）
- [x] 制作"常见报错排查"速查表（学生最容易遇到的问题）
- [x] 添加"术语表"页面（25+ 核心术语的通俗解释）

---

## 第四阶段：互动增强（v2.0） 🔶 基本完成

**目标**：从静态文档进化为互动学习平台。

- [x] 嵌入在线代码预览组件（iframe srcdoc），支持在网页内直接查看运行效果
- [x] 为每节课添加小测验（选择题），巩固知识点（共 28 道题）
- [x] 添加"学生作品展示墙"页面，收录优秀学生作品
- [x] 接入评论系统（Giscus），基于 GitHub Discussions 的留言讨论
- [ ] 制作关键步骤的 GIF 动图或短视频演示（需录制）

---

## 第五阶段：内容扩展（v3.0） 📋 待启动

**目标**：丰富课程体系，覆盖更多学习路径。

- [ ] **进阶篇**（第8-12课）：Python 入门、数据可视化、简易聊天机器人等
- [ ] **专题篇**：针对特定兴趣的独立教程（如"用 AI 做像素画"、"用 AI 写小说"）

- [ ] 多语言支持（英文版），扩大受众范围

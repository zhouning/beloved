# 常见问题 FAQ

> 遇到问题别着急，看看这里有没有你需要的答案！

## 环境搭建

### VS Code 安装后打不开？

试试重启电脑，如果还不行，可以去 [VS Code 官网](https://code.visualstudio.com/) 重新下载安装。

### 终端在哪里打开？

在 VS Code 中，按快捷键 `` Ctrl + ` ``（键盘左上角那个反引号键），终端就会出现在下方。也可以点击菜单栏 **终端 → 新建终端**。

### 输入命令后提示"不是内部或外部命令"？

这通常意味着对应的软件没有正确安装，或者没有添加到系统路径。建议重新按照[课前准备](/prepare)的步骤安装一遍。

## Claude Code 相关

### Claude Code 没有反应？

1. 检查网络连接是否正常
2. 确认已经正确安装：在终端输入 `claude --version` 查看版本号
3. 如果安装了但没反应，试试关掉终端重新打开

### Claude Code 生成的代码有错误怎么办？

这很正常！AI 也会犯错。你可以：
- 把错误信息直接复制给 Claude Code，告诉它"报错了，帮我修一下"
- 用更详细的描述重新告诉它你想要什么
- 参考第5课学到的 debug 思维来定位问题

### 提示词写了但效果不好？

写好的提示词需要练习，试试这些技巧：
- **说清楚**：不要只说"做个网页"，而是说"做一个蓝色背景的个人介绍网页，上面有我的名字和爱好"
- **分步骤**：把一个大需求拆成几个小步骤，一步一步来
- **给例子**：告诉 AI 你想要的效果像什么样子

## 网页和代码

### 怎么在浏览器里打开做好的网页？

在文件管理器中找到你的 `.html` 文件，双击就能在浏览器中打开。也可以在 VS Code 中右键文件，选择"在默认浏览器中打开"。

### 修改了代码但浏览器没变化？

记得保存文件（`Ctrl + S`），然后在浏览器中刷新页面（`Ctrl + R` 或 `F5`）。

### 网页在手机上看起来不一样？

这是因为手机屏幕比电脑小。可以让 Claude Code 帮你"让这个网页在手机上也好看"，它会添加响应式设计的代码。

不需要！只要会打字就行。课程从最基础的概念开始教起。

### 做完7节课之后学什么？

可以看第7课末尾的"下一步"建议，那里列出了很多继续学习的方向，比如学 Python、做更复杂的游戏等等。

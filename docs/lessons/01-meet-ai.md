# 第1课：认识AI与Claude Code 🤖

> 欢迎来到AI编程的奇妙世界！在这节课里，你会认识一个超级厉害的新朋友——它能听懂你说的话，还能帮你写代码！

## 学习目标

学完这节课，你将能够：

- 理解什么是AI（人工智能），它在日常生活中无处不在
- 认识Claude Code——你的AI编程搭档
- 学会打开终端，和Claude Code对话
- 掌握几个好用的"魔法咒语"（提示词）

## 什么是AI？其实你早就认识它了！

AI就是**人工智能**（Artificial Intelligence）的缩写。听起来很高大上对不对？但其实你每天都在和AI打交道呢！

想想看这些场景：

- **Siri / 小爱同学**：你对着手机说"明天天气怎么样？"，它能听懂并且回答你——这就是AI！
- **抖音 / B站推荐**：为什么你刷到的视频都是你喜欢看的？因为AI在背后观察你的喜好，猜你想看什么。
- **美颜滤镜**：拍照的时候那些可爱的猫耳朵特效，AI能识别你的脸在哪里，然后精准地把特效贴上去。
- **拼音输入法**：你打几个字母，它就能猜出你想输入什么词——这也是AI在帮忙！

::: tip 小贴士
AI并不是什么神秘的东西。你可以把它想象成一个**超级勤奋的学生**——它读了互联网上几乎所有的文章和书，记住了超多知识，所以你问它问题的时候，它能给出不错的回答。
:::

简单来说，AI就是人类教会计算机"思考"和"学习"的技术。它不是真的有脑子，但是它可以通过学习大量数据来模仿人类的思考方式。

## 认识你的新朋友：Claude Code

**Claude Code** 是一个AI编程助手。你可以把它想象成一个**坐在你旁边的编程高手朋友**：

- 它**听得懂中文**！你用中文跟它说话就行，完全不用担心英语
- 你告诉它你想做什么，它就能帮你写出代码
- 你可以让它改东西、解释代码、甚至帮你想点子
- 它不会嘲笑你的问题，也不会不耐烦——你问多少次都可以

::: info 魔法咒语
和Claude Code对话就像和朋友聊天一样自然。比如你可以说：

```
你好！你能做什么？
```

它就会告诉你它能帮你做哪些事情。就是这么简单！
:::

**Claude Code和普通聊天AI的区别是什么？**

普通的AI聊天（比如网页版）只能跟你文字对话。但是Claude Code更厉害——它可以直接在你的电脑上**创建文件、写代码、运行程序**。它不只是"说"，它还能"做"！

## 动手时间：跟Claude Code打个招呼吧！

好了，理论讲够了，让我们开始动手吧！

### 第一步：打开终端

**终端**（Terminal）是什么？你可以把它想象成一个"用文字和电脑对话的窗口"。平时你用鼠标点来点去操作电脑，而终端让你用**打字**的方式告诉电脑该做什么。

在Windows上打开终端：
1. 按下键盘上的 `Win` 键（就是那个Windows图标的键）
2. 输入 `终端` 或者 `Terminal`
3. 点击打开它

你会看到一个黑色（或深色）的窗口，里面有一个闪烁的光标在等你输入——这就是终端啦！

### 第二步：启动Claude Code

在终端里输入：

```bash
claude
```

然后按下回车键（Enter）。

稍等几秒钟……Claude Code就启动了！你会看到它在跟你打招呼。

### 第三步：开始对话！

现在你可以跟它聊天了！试试输入下面这些内容：

**让它写一首诗：**
```
帮我写一首关于春天的短诗，要押韵哦
```

**让它讲个笑话：**
```
给我讲一个适合中学生听的冷笑话
```

**让它做数学题：**
```
一个水池有两个水管，A管单独注满要6小时，B管单独注满要4小时，两个一起开多久能注满？
```

::: tip 小贴士
如果你想退出Claude Code，可以输入 `/exit` 然后按回车，就可以回到普通的终端了。
:::

## 魔法咒语大全

在AI的世界里，你跟AI说的话叫做**提示词**（Prompt）。好的提示词就像好的魔法咒语——说得越清楚，AI就越能给你满意的结果。

下面是一些你可以直接复制粘贴到Claude Code里试试的魔法咒语：

::: info 魔法咒语
**咒语1：创意写作**
```
帮我编一个短故事，主角是一只会编程的猫，它用代码解决了一个大问题
```

**咒语2：知识问答**
```
用简单的语言解释一下，网站是怎么工作的？就像跟13岁的朋友解释一样
```

**咒语3：翻译助手**
```
把"我正在学习编程，这太有趣了！"翻译成英语、日语和法语
```

**咒语4：头脑风暴**
```
我想做一个有趣的小网页，你能给我5个创意点子吗？要适合初学者做的
```
:::

每个咒语输入之后按回车，然后等Claude Code回复。认真看看它的回答，是不是很有意思？

## 重要概念

让我们来总结一下今天学到的几个重要概念：

### AI（人工智能）
计算机通过学习大量数据来模仿人类思考的技术。它不是真的"聪明"，而是因为学过太多东西，所以能给出有用的回答。

### Prompt（提示词）
你跟AI说的话就叫提示词。就像你跟朋友说话一样，你说得越清楚，AI就越能理解你想要什么。

### Terminal（终端）
用文字和电脑对话的窗口。程序员们天天用它来工作。现在你也会用了！

::: details 想了解更多？
**AI是怎么学习的？**

想象一下，如果你读了世界上所有的书、看了所有的文章，你是不是也能回答很多问题？AI就是这样——它被"喂"了互联网上海量的文字资料，通过分析这些文字之间的规律来学习。

比如它发现"太阳从____升起"这个空格里，大家最常填"东方"，所以它就学会了这个知识。当然实际过程要复杂得多，但基本原理就是这样：**从大量数据中找规律**。

这就是为什么AI有时候也会说错话——它不是真正"理解"了知识，它只是在做非常高级的"猜测"。
:::

恭喜你完成了第一课的学习！现在是挑战时间。

**你的任务是：** 自己想一个创意问题，问问Claude Code！

这个问题可以是任何你感兴趣的东西，比如：
- 让它帮你设计一个游戏的故事背景
- 问它某个你好奇的科学问题
- 让它帮你写一段自我介绍
- 或者任何你能想到的好玩的事情！

::: warning 注意
AI很强大，但它不是万能的。有时候它可能会给出不太准确的信息，特别是涉及到最新新闻或者非常专业的知识时。所以对于重要的事情，记得找其他来源确认一下哦！
:::

**关键原则：不要害怕尝试！** 你不会把电脑弄坏的。Claude Code是一个安全的环境，你可以尽情探索、随便尝试。

## 提交你的作品

做出了满意的作品？来展示给大家看吧！

**提交方式**：在 [GitHub Discussions](https://github.com/zhouning/beloved/discussions) 发帖，包含以下信息：

1. **作品名称**
2. **你的名字**（可以用昵称）
3. **对应课程**（第几课的项目）
4. **简单介绍**（1-2句话描述你的作品有什么特别的）
5. **截图或链接**（如果有的话）

::: tip 小贴士
不用担心作品不够"完美"——每一个你亲手做出来的项目都值得骄傲！我们更看重创意和努力，而不是代码有多复杂。
:::

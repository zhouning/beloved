# AI编程启蒙课

> 用 Claude Code 开启你的编程之旅 —— 写给初中生的零基础 AI 编程教程

不需要任何编程基础，用自然语言和 AI 一起做出酷炫的小应用！

**在线学习：[ai4junior.com](https://www.ai4junior.com/)**

## 课程内容

### 入门篇：网页开发（第1-7课）

从零开始，用 AI 做出你的第一个网页和小应用：

| 课程 | 你会做出什么 | 学到什么 |
|------|-------------|---------|
| 第1课 | 和 AI 聊天 | 认识 AI，学会对话 |
| 第2课 | 个人介绍网页 | 向 AI 描述需求 |
| 第3课 | 趣味性格测试 | 拆解任务的思维 |
| 第4课 | 学习计时器 | 迭代改进的方法 |
| 第5课 | 创意小游戏 | 调试和解决问题 |
| 第6课 | AI故事生成器 | 理解 AI 的工作原理 |
| 第7课 | 个人作品集 | 完整项目流程 |

### 进阶篇：Python 编程（第8-12课）

学会 Python，解锁数据分析、可视化和聊天机器人：

| 课程 | 你会做出什么 | 学到什么 |
|------|-------------|---------|
| 第8课 | 文字冒险游戏 | Python 基础语法 |
| 第9课 | 成绩分析报告 | 数据结构与文件操作 |
| 第10课 | 数据可视化图表 | 图表生成与数据分析 |
| 第11课 | 聊天机器人 | API 调用与对话逻辑 |
| 第12课 | 毕业设计作品 | 完整项目开发流程 |

### 趣味专题

| 专题 | 内容 |
|------|------|
| 像素画编辑器 | 用代码画像素画 |
| 表情包生成器 | 自制搞笑表情包 |
| 音乐节拍机 | 用代码玩音乐 |

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run docs:dev

# 构建生产版本
npm run docs:build
```

## 技术栈

- [VitePress](https://vitepress.dev/) — 静态站点生成器
- [Claude Code](https://claude.ai/code) — AI 编程助手
